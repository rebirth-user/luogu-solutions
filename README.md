# luogu-solutions

洛谷刷题代码仓库

## 文件结构

```
├── 数学/           # 数学相关题目
├── 动态规划/       # DP相关题目
├── 图论/           # 图论相关题目
├── 数据结构/       # 数据结构相关题目
├── 搜索/           # 搜索相关题目
├── 贪心/           # 贪心算法题目
├── 二分/           # 二分相关题目
├── 字符串/         # 字符串处理题目
├── 模拟/           # 模拟题
└── 其他/           # 未分类题目
```

## 文件命名规范

- 格式：`P{题号}_{题名简称}.cpp`
- 示例：`P1001_A+B_Problem.cpp`

## 提交规范

每次提交信息格式：
```
添加 P{题号} {题名}
```
